Konsep Inti

Templat

Dalam aplikasi angular, anda memindahkan pekerjaan mengisi data templat laman dari (dahulunya) server ke klien. Hasilnya sistem berstruktur lebih baik untuk pemutakhiran laman secara dinamis. Di bawah ini adalah fitur inti yang akan digunakan.

  • Pengikatan data

  • Ekspresi

  • Direcktif

  • Pemandangan dan rute (lihat contoh)

  • Saringan

  • Formulir dan Konsep Formulir AngularJS

Struktur Aplikasi

  • Pos CatatanDaring (Blog): Kapan menggunakan direktif, kontroler atau layanan

  • Mengawatkan Aplikasi: Injeksi Dependensi (Suntik Keterkaitan)

  • Menyingkapkan model ke templat: Lingkup

  • Menghubungkan dengan server: $http, $resource

Fitur AngularJS lainnya

  • Animasi: Konsep Inti, API ngAnimate, dan Animasi dalam AngularJS versi 1.2

  • Keamanan: Lucutan Kontekstual Ketat (Strict Contextual Escaping), Kebijakan Isi Keamanan (Content Security Policy), $sanitize, video

  • Internasionalisasi dan Lokalisasi: Panduan Angular untuk i18n dan l10n, date filter, currency filter, Membangun dukungan multilingual

  • Aplikasi Mobil: acara ngTouch

Pengujian

  • Unit pengujian: Menggunakan Karma (video), Unit pengujian, Pengujian layanan, Karma dalam Webstorm

  • Pengujian Skenario: Busur derajat

Pokok pembicaraan spesifik

  • Masuk-catatan (login): contoh Google, contoh Facebook, siasat penyahihan, otorisasi gaya-unix

  • Aplikasi Mobil: Angular dalam Panduan Aplikasi Mobil, PhoneGap

  • Bahasa Pemrograman Lain: CofeeScript, Dart

  • Waktu-aktual (realtime): Socket.io, OmniBinder

  • Visualisasi: SVG, D3.js

Perkakas

  • Pengawakutuan (debugging): Batarang

  • Pengujian: Karma, Protractor

  • Editor dukungan: Webstorm (dan video), Sublime Text, Visual Studio

  • Alirkerja: Yeoman.io dan Bimbingan Yeoman Angular

Kelengkapan Pustaka

Ini adalah daftar pendek pustaka dengan spesifik dukungan dan dokumentasi untuk menggunakan Angular. Anda bisa menemukan daftar lengkap dari luar pustaka Angular yang diketahui di ngmodules.org.

  • Internasionalisasi: angular-translate, angular-gettext

  • Layanan REST: Restangular

  • Pengujung (backend) SQL dan NoSQL: BreezeJS, AngularFire

  • Gawai UI: KendoUI, UI Bootstrap, Wijmo
