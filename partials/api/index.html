Dokumentasi AngularJS API

Selamat datang di laman dokumentasi AngularJS API. Laman ini berisi materi rujukan AngularJS untuk versi .

Dokumentasi terbagi dalam modul-modul yang berisi berbagai komponen dari aplikasi AngularJS. Yang termasuk komponennya adalah direktif, layanan, saringan, penyedia, tipe, API global dan samaran pengujian.

GugusanNama Angular $ dan $$ Untuk mencegah benturan nama yang tak diinginkan saat pengkodean, Angular memberi prefiks pada objek publik dengan $ dan nama privat object dengan $$. Tolong jangan gunakan prefiks $ atau $$ pada pengkodean.

GugusanNama Angular

ng (modul inti)

Modul ini disediakan secara praseleksi dan mengandung komponen inti dari AngularJS.

Direktif

Ini adalah koleksi inti dari direktif yang dapat digunakan dalam kode templat untuk membangun sebuah aplikasi AngularJS.

Beberapa contoh termasuk: ngClick, ngInclude, ngRepeat, dsb…

Layanan / Pabrikasi

Ini adalah koleksi inti dari layanan yang digunakan dalam aplikasi DI ( Injeksi Dependensi ).

Beberapa contoh termasuk: $compile, $http, $routeParams, $location, dsb…

Saringan

Saringan inti tersedia di ng module yang digunakan untuk mengubah data templat sebelum ditampilkan dalam ekspresi dan direktif.

Beberapa contoh termasuk: filter, date, currency, lowercase, uppercase, dsb...

API Global

Fungsi inti API global terikat pada objek angular. funcgsi inti ini berguna untuk operasi Javaskrip taraf rendah dalam aplikasi.

Beberapa contoh termasuk: angular.copy(), angular.equals(), angular.element(), dsb...

ngRoute

Gunakan ngRoute untuk mengarahkan rute URL ke aplikasi. Modul ngRoute mendukung manajemen URL baik lewat tandapagar (# - hashbang) dan pushState HTML5.

Sertakan fail angular-route.js dan set ngRoute sebagai sebuah dependensi untuk membuatnya bekerja dalam aplikasi.
Layanan / Pabrikasi Layanan berikut digunakan untuk manajemen rute:
  • $routeParams digunakan untuk mengambil isi untaian-pertanyaan (querystring) dalam URL.
  • $route digunakan untuk mengambil rincian rute yang sedang diakses.
  • $routeProvider digunakan untuk pencatatan rute dalam aplikasi.
Direktif Direktif ngView akan menampilkan templat dari rute terkini dalam laman tersebut.

ngAnimate

Gunakan ngAnimate untuk menyalakan fitur animasi ke dalam aplikasi. Berbagai ngDirective inti akan menyediakan kait animasi ke dalam aplikasi jika ngAnimate disertakan. Kemudian animasi dirumuskan dengan menggunakan animasi/transisi CSS atau panggilbalik (callback) Javaskrip.

Sertakan fail angular-route.js dan set ngRoute sebagai sebuah dependensi untuk membuatnya bekerja dalam aplikasi.
Layanan / Pabrikasi Use $animate to trigger animation operations within your directive code.
CSS-based animations Follow ngAnimate’s CSS naming structure to reference CSS transitions / keyframe animations in AngularJS. Once defined the animation can be triggered by referencing the CSS class within the HTML template code.
JS-based animations Use module.animation() to register a JavaScript animation. Once registered the animation can be triggered by referencing the CSS class within the HTML template code.

ngResource

Use the ngResource module when querying and posting data to a REST API.

Include the angular-resource.js file and set ngResource as a dependency for this to work in your application.
Layanan / Pabrikasi The $resource service is used to define RESTful objects which communicate with a REST API.

ngCookies

Use the ngCookies module to handle cookie management within your application.

Include the angular-cookies.js file and set ngCookies as a dependency for this to work in your application.
Layanan / Pabrikasi The following services are used for cookie management:
  • The $cookie service is a convenient wrapper to store simple data within browser cookies.
  • $cookieStore is used to store more complex data using serialization.

ngTouch

Use ngTouch when developing for mobile browsers/devices.

Include the angular-touch.js file and set ngTouch as a dependency for this to work in your application.
Layanan / Pabrikasi The $swipe service is used to register and manage mobile DOM events.
Direktif Various Direktif are available in ngTouch to emulate mobile DOM events.

ngSanitize

Use ngSanitize to securely parse and manipulate HTML data in your application.

Include the angular-sanitize.js file and set ngSanitize as a dependency for this to work in your application.
Layanan / Pabrikasi The $sanitize service is used to clean up dangerous HTML code in a quick and convenient way.
Saringan The linky filter is used to turn URLs into HTML links within the provided string.

ngMock

Use ngMock to inject and mock modules, factories, services and providers within your unit tests

Include the angular-mocks.js file into your test runner for this to work.
Layanan / Pabrikasi

ngMock will extend the behavior of various core services to become testing aware and manageable in a synchronous manner.

Some examples include: $timeout, $interval, $log, $httpBackend, etc...

API Global

Various helper functions are available to inject and mock modules within unit test code.

Some examples inject(), module(), dump(), etc...
