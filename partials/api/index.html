Dokumentasi API AngularJS

Selamat datang di laman dokumentasi API AngularJS. Laman ini berisi materi rujukan AngularJS untuk versi .

Dokumentasi terbagi dalam modul-modul yang berisi berbagai komponen dari aplikasi AngularJS. Yang termasuk komponennya adalah direktif, layanan, saringan, pemasok, tipe, API global dan samaran pengujian.

GugusanNama Angular $ dan $$ Untuk mencegah benturan nama yang tak diinginkan saat pengkodean, Angular memberi prefiks pada objek publik dengan $ dan nama privat object dengan $$. Tolong jangan gunakan prefiks $ atau $$ pada pengkodean.

GugusanNama Angular

ng (modul inti)

Modul ini disediakan secara praseleksi (default) dan mengandung komponen inti dari AngularJS.

Direktif

Ini adalah koleksi inti dari direktif yang dapat digunakan dalam kode templat untuk membangun sebuah aplikasi AngularJS.

Beberapa contoh termasuk: ngClick, ngInclude, ngRepeat, dsb…

Layanan / Pabrikasi

Ini adalah koleksi inti dari layanan yang digunakan dalam aplikasi DI ( Injeksi Dependensi ).

Beberapa contoh termasuk: $compile, $http, $routeParams, $location, dsb…

Saringan

Saringan inti tersedia di modul ng (ng module) yang digunakan untuk mengubah data templat sebelum ditampilkan dalam bentuk ekspresi dan direktif.

Beberapa contoh termasuk: filter, date, currency, lowercase, uppercase, dsb...

API Global

Fungsi inti API global terikat pada objek angular. fungsi inti ini berguna untuk operasi Javaskrip taraf rendah (low-level Javascript) dalam aplikasi.

Beberapa contoh termasuk: angular.copy(), angular.equals(), angular.element(), dsb...

ngRoute

Gunakan ngRoute untuk mengarahkan rute URL ke aplikasi. Modul ngRoute mendukung manajemen URL baik lewat tandapagar (# - hashbang) dan pushState HTML5.

Sertakan fail angular-route.js dan set ngRoute sebagai sebuah dependensi untuk membuatnya bekerja dalam aplikasi.
Layanan / Pabrikasi Layanan berikut digunakan untuk manajemen rute:
  • $routeParams digunakan untuk mengambil isi untaian-pertanyaan (querystring) dalam URL.
  • $route digunakan untuk mengambil rincian rute yang sedang diakses.
  • $routeProvider digunakan untuk mencatatkan rute dalam aplikasi.
Direktif Direktif ngView akan menampilkan templat dari rute terkini dalam laman tersebut.

ngAnimate

Gunakan ngAnimate untuk menyalakan fitur animasi ke dalam aplikasi. Berbagai ngDirective inti akan menyediakan kait animasi ke dalam aplikasi jika ngAnimate disertakan. Kemudian animasi dirumuskan dengan menggunakan animasi/transisi CSS atau panggilbalik (callback) Javaskrip.

Sertakan fail angular-animate.js dan set ngAnimate sebagai sebuah dependensi untuk membuatnya bekerja dalam aplikasi.
Layanan / Pabrikasi Gunakan $animate untuk memicu operasi animasi di dalam pengkodean direktif.
Animasi berbasis-CSS Ikuti struktur penamaan CSS ngAnimate ketika merujuk pada animasi transisi/lis-utama (keyframe) di AngularJS. Sekali dirumuskan animasi dapat dipicu dengan merujuk kelas CSS di dalam pengkodean templat HTML.
Animasi berbasis-JS Gunakan module.animation() untuk mendaftarkan animasi Javaskrip. Sekali dirumuskan animasi dapat dipicu dengan merujuk kelas CSS di dalam pengkodean templat HTML.

ngResource

Gunakan modul ngResource ketika menanyakan dan mengepos data ke API REST.

Sertakan fail angular-resource.js dan set ngResource sebagai sebuah dependensi untuk membuatnya bekerja dalam aplikasi.
Layanan / Pabrikasi Layanan $resource digunakan untuk merumuskan objek REST saat berkomunikasi dengan API REST.

ngCookies

Gunakan modul ngCookies untuk mengurus manajemen kuki dalam aplikasi.

Sertakan fail angular-cookies.js dan set ngCookies sebagai sebuah dependensi untuk membuatnya bekerja dalam aplikasi.
Layanan / Pabrikasi Layanan berikut digunakan untuk manajemen kuki:
  • Layanan $cookie adalah pembungkus yang mudah untuk menyimpan sedikit data dalam kuki peramban.
  • $cookieStore digunakan untuk menyimpan data yang lebih rumit menggunakan serialisasi (serialization).

ngTouch

Gunakan ngTouch saat mengembangkan peranti/peramban untuk aplikasi mobil.

Sertakan fail angular-touch.js dan set ngTouch sebagai sebuah dependensi untuk membuatnya bekerja dalam aplikasi.
Layanan / Pabrikasi Layanan $swipe digunakan untuk pencatatan dan mengatur acara MOD (DOM events) aplikasi mobil.
Direktif Berbagai direktif sudah tersedia dalam ngTouch untuk meniru acara MOD aplikasi mobil.

ngSanitize

Gunakan ngSanitize untuk memilah dan merekayasa data HTML secara aman dalam aplikasi.

Sertakan fail angular-sanitize.js dan set ngSanitize sebagai sebuah dependensi untuk membuatnya bekerja dalam aplikasi.
Layanan / Pabrikasi Layanan $sanitize digunakan untuk membersihkan bahaya kode HTML dengan mudah dan cepat.
Saringan Saringan tautan (linky filter) digunakan untuk mengubah URL menjadi taut HTML dalam untaian (string) yang disediakan.

ngMock

Gunakan ngMock untuk memasukkan dan menyamar sebagai modul, pabrikan, layanan dan pemasok dalam unit uji.

Sertakan fail angular-mocks.js ke dalam pekerja tes (test runner) untuk membuatnya bekerja dalam aplikasi.
Layanan / Pabrikasi

ngMock akan menambah kemampuan berbagai layanan inti sehingga membuat aplikasi sadar-akan-keterujian (testing aware) dan mudah diatur secara sinkron.

Beberapa contoh termasuk: $timeout, $interval, $log, $httpBackend, dsb...

API Global

Berbagai fungsi pembantu yang tersedia untuk memasukkan dan menyamar sebagai modul dalam pengkodean unit uji.

Beberapa contoh: inject(), module(), dump(), dsb...
